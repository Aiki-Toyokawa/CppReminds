コンストラクタ

クラスでオブジェクトを作成する際に様々な初期化が必要となる
その初期化処理をコンストラクタと呼ばれるメンバ関数で行う(クラス内で宣言する)
クラスからオブジェクトが作成されるときに自動的に呼び出される


クラス名を関数名として使う
コンストラクタは戻り値がない、voidも記述しない

定義)
クラス名::クラス名(引数のリスト){
   ...
}

例)
class Car {
private:
	int num;
	double gas;
public:
	Car();
};

Car::Car() {       //自動的に呼び出される
	num = 0;
	gas = 0.0;
	std::cout << "made Car";
}

応用)
コンストラクタのオーバーロード
